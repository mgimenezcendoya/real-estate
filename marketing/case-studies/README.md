# Case Studies

Esta carpeta se llena a partir del mes 2, cuando los primeros clientes piloto tengan resultados medibles.

## Template de case study

```markdown
# [Nombre de la desarrolladora] — Case Study

## El cliente
- Desarrolladora: [nombre]
- Proyectos: [cuántos, dónde]
- Equipo comercial: [tamaño]
- Desafío principal: [en sus palabras]

## Antes de Realia
- Tiempo promedio de respuesta: [X horas]
- Leads perdidos por semana: [X estimado]
- Cómo gestionaban obra/inversores: [manual, Excel, WA]

## Después de Realia
- Tiempo de respuesta: [X segundos]
- Leads atendidos 24/7: [X%]
- Mejora en conversión: [X%]
- Horas ahorradas por semana: [X]

## Quote del cliente
> "[testimonio directo]"
> — [Nombre, Cargo]

## Resultado clave (para usar en posts)
"[Dato más impactante en una línea]"
```

## Cómo conseguir el case study

1. A las 2 semanas de piloto, pedir métricas: "¿Cuántos leads atendió el bot? ¿Cómo se compara con antes?"
2. Pedir permiso para publicar con nombre
3. Si no quieren nombre, usar anónimo: "Una desarrolladora con 3 proyectos en CABA..."
